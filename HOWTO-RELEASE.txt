To make toolbox and package

  cmake -B BUILD -S .
  make -C BUILD package test

packages are in

 BUILD/dist/geographiclib*.{tar.gz,zip}

To install code

  make -C BUILD install

This installs in ${CMAKE_INSTALL_PREFIX}/${INSTDIR}/geographiclib
